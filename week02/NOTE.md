# 每周总结可以写在这里

## 1.产生式

	#### BNF

1. 用尖括号括起来的名称来表示语法结构名

2. 语法结构分成基础结构和需要用其他语法结构定义的复合结构
   1. 基础结构称 终结符
   2. 复合结构称 非终结符
3. 引号和中间的字符表示终结符
4. 可以有括号 代表终结符，非终结符的需要"(" ")"
5. *表示可以重复多次
6. |表示或
7. +表示至少一次
